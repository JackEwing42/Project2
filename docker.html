Docker Cheat Sheet

Basic Docker Commands & Dockerfile Reference

Basic Docker Commands

  • docker --version - Show Docker version
  • docker build -t <name> . - Build image from Dockerfile
  • docker run <image> - Run a container
  • docker ps - List running containers
  • docker ps -a - List all containers (including stopped)
  • docker stop <container_id> - Stop a container
  • docker rm <container_id> - Remove a container
  • docker rmi <image_id> - Remove an image
  • docker images - List images
  • docker exec -it <container_id> bash - Access a running container

Common Dockerfile Instructions

# Start from a base image
FROM node:14

# Set working directory
WORKDIR /app

# Copy files
COPY package*.json ./

# Install dependencies
RUN npm install

# Copy app source
COPY . .

# Expose port
EXPOSE 3000

# Start the app
CMD ["node", "app.js"]

Instruction Summary:

  • FROM - Base image
  • WORKDIR - Working directory in the container
  • COPY - Copy files from host to image
  • RUN - Run commands during build
  • EXPOSE - Declare the port the container listens on
  • CMD - Default command to run on container start
